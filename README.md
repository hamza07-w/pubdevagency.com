# pubdev-agency
my Startup website, built with ReactJs &amp; TailwindCss

Welcome to the official repository for pubdev agency website! This is aReact JS project is designed to publish our open sources in order to extend our community.

## Installation
To get started with [Your Project Name], follow these steps:

Clone the repository:
```bash
git clone https://github.com/hamza07-w/pubdevagency.com.git
```

Navigate to the project directory:
```bash
cd pubdev-agency
```

Install dependencies:
```bash
npm install
```

Running the Application
```bash
npm run start
```

This will start the development server, and you can view the app by navigating to http://localhost:3000 in your web browser.
